Detailed Description

DESCRIPTION

This class holds two functions. One to check if a player is in stale mate, and one to check if a player is in check mate. This class has two vectors. The vector checkTiles holds a vector of pieces that put the king in check. The mustGoTiles holds every location on the board that the opposing player can move one of their pieces to get out of check. These vectors are used to find if a player is in check mate
