\subsection{Detailed Description}
\hypertarget{classvalidMoveTest_DESCRIPTION}{}\subsection{D\-E\-S\-C\-R\-I\-P\-T\-I\-O\-N}\label{classvalidMoveTest_DESCRIPTION}
This class holds all the functions and data for both players Kings. It sees if a king can move, and sees if a king is in check. It holds functions that checks if a move for a king is valid. Also does all the handling for check scenarios.
